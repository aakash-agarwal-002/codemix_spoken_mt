# Codemix Spoken MT

## Project Overview
Codemix Spoken MT is an innovative project designed to facilitate spoken language translation. With the rapid global interaction, this project aims to bridge communication gaps by providing users with accurate translations in real-time.

## Features
- **Real-time Translation**: Supports live translation for multiple spoken languages.
- **User-friendly Interface**: Easy to navigate interface for quick access to translation features.
- **High Accuracy**: Utilizes advanced algorithms for precise translations.
- **Multi-platform Support**: Compatible with various devices and operating systems.
- **Customizable Settings**: Users can personalize their experience by adjusting settings to fit their needs.

## Installation Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/aakash-agarwal-002/codemix_spoken_mt.git
   cd codemix_spoken_mt
   ```
2. **Install Dependencies**:
   Make sure to have Node.js installed. Then run:
   ```bash
   npm install
   ```
3. **Run the Application**:
   Start the application with:
   ```bash
   npm start
   ```

## Usage
- Launch the application after installation.
- Select the languages you wish to translate.
- Click the microphone icon to start speaking and receive real-time translations.
- You can customize settings according to your preferences.

## Dependencies
- **Node.js**: v14 or higher
- **Express**: For the backend server
- **Socket.io**: For real-time communication
- **TensorFlow.js**: For machine learning capabilities
- **Other Related Libraries**: Please refer to `package.json` for more details.
